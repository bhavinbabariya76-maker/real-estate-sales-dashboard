# real-estate-sales-dashboard
A Python-based real estate sales dashboard analyzing sales performance, inventory, revenue, conversion rates, and marketing ROI using Pandas and Matplotlib.
Project Overview

This project analyzes real estate sales data to help developers understand sales performance, inventory trends, marketing effectiveness, and revenue generation.
The dashboard was built using Python and focuses on transforming raw sales data into actionable business insights.

Objectives

* Analyze monthly sales trends
* Monitor inventory availability
* Calculate conversion rates from leads to bookings
* Evaluate marketing return on investment (ROI)
* Identify top-performing projects and cities
* Generate business insights using data visualization

Tools and Technologies

* Python
* Pandas
* Matplotlib
* Jupyter Notebook
* CSV Data Files

Key Metrics Calculated

* Revenue
* Conversion Rate
* Inventory Remaining
* Marketing ROI
* Units Sold
* Average Price per Square Foot

Visualizations Included

* Monthly Sales Trend
* Sales by Project
* Marketing Spend vs Bookings
* Conversion Rate by Project
* Price Trend Analysis
